Sobrecarga (de funções e de operadores)



Sobre carga de funções é uma alternativa a templates a diferença entre eles é que sobrecargas de funçoes é limitada ao uso de tipos de dados ou tipos de parâmetros. A sobre carga de funções exige que para cada possíveis tipos de dados de entrada seja uma função.

Diferentemente de um template que com apenas uma função template é possível combinar tipos de dados de entrada. Mas não deixa de ser um ótimo recurso ou técnica de programação. Veja o exemplo a baixo. 
                    
                        #include <iostream>
                        using namespace std;

                        
                        int      soma(int, int);      //Função soma que recebe dois inteiros
                        double   soma(double, double);//Função soma que recebe dois decimais
                        int      soma(int, int, int); //Função soma que recebe três inteiros


                        int main()
                        {
                            cout << "Hello REAA!"<< endl;
                            cout<<"soma com 2 int:"<<soma(2, 3)    <<endl;//Retorna a soma de dois 
                            cout<<"soma com 3 int:"<<soma(2, 3, 4) <<endl;//Retorna a soma de três inteiros
                            cout<<"soma com 2 double:"<<soma(8.2, 1.9)<<endl;//Retorna a soma de dois decimais

                            return 0;
                        }

                        //sobrecarga de dois inteiros
                        int      soma(int A, int B)
                        {
                            return A+B;
                        }
                        //sobrecarga de dois double

                        double   soma(double A, double B)
                        {
                            return A+B;
                        }
                        //sobrecarga de três inteiros

                        int      soma(int A, int B, int C)
                        {
                            return A+B+C;
                        }
