Processamento de Arquivos .txt e .csv



Essa parte de Processamento ou manipulação de arquivos, digo eu que é a parte mais prazerosa desta jornada. Pois é possível armazenar dados mesmo depois de fechar o programa. Também é possivél leros dados armazenados nele.

A sintaxe da manipulação de arquivos é o que segue a baixo, e com esta pequena construção você pode criar outros arranjos de programa.

Primeiramente, vou começar armazenando num arquivo .txt, que é um bloco qualquer, depois vou armazenas num arquivo .csv que é a planilha do Exel. 

                    
                            #include <iostream>
                            using namespace std;
                            
                            #include <cstdlib>
                            using std::exit;
                            
                            #include <iomanip>
                            using std::setw;
                            
                            #include <fstream>
                            using std::ofstream;//para trabalhar com arquivos
                            
                            
                            int main()
                            {
                                //Boas vindas
                                cout << "Hello REAA!" << endl;
                            
                                float resultado=60;
                                //Gera o arquivo se caso não existir.
                                ofstream Arquivo("Arquivo.txt", ios::out);
                                //Verifica se o arquivo foi aberto com exidô
                                if(!Arquivo)
                                {
                                    cout << "Sem sucesso ao abrir o arquivo:" << endl;
                                    exit(1);//Aborta o programa, se caso não rolar.
                                }
                            
                                cout << "Arquivo aberto com sucesso: " << endl;
                            
                                Arquivo <<resultado << setw(3) << "Hz" << "Padrão Brazil." <<endl;
                                Arquivo << "======================";
                            
                                Arquivo.close();//Fecha o arquivo
                                while(1);
                                return 0;
                            }

Usamos o objeto ofstream para especificar a saida de dados indo direto para um arquivo, ou seja, Out File Stream
